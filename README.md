# Wayland Mirror GUI

[🇺🇸 English](README_EN.md) | [🇺🇦 Українська](README.md)

Проста, але дуже потужна графічна утиліта для дублювання екранів (mirroring) на Wayland. Створена спеціально для тих, кому не вистачає зручного "Windows-like" інтерфейсу для налаштування кількох моніторів.

## Особливості
* **Нативне дублювання (Zero-copy):** Використовує `wlr-randr` для ідеального попіксельного масштабування без жодної втрати продуктивності чи затримок.
* **Дублювання у вікні:** Інтеграція з `wl-mirror` для виведення екрана у зручне вікно (корисно для стрімінгу або прев'ю).
* **Сучасний та компактний інтерфейс:** Вибір екранів захований у зручні випадаючі списки з чекбоксами — ідеальна симетрія та нічого зайвого на екрані.
* **Мульти-моніторність:** Можливість транслювати один екран одразу на декілька телевізорів чи проекторів.
* **"Розумні" чекбокси:** Миттєве включення або відключення трансляції на льоту просто зняттям галочки — без потреби зупиняти всю сесію.
* **Двомовність та пам'ять:** Програма підтримує англійську та українську мови. Вибрана мова зберігається автоматично (за замовчуванням — англійська, щоб бути зрозумілою для всіх).
* **Розумна пам'ять позицій:** Перед дублюванням програма запам'ятовує точні координати зовнішнього монітора. При скасуванні дублювання вона автоматично повертає його рівно на те саме місце.
* **Автовизначення:** Програма автоматично бачить, які екрани зараз здубльовані (включаючи вікна `wl-mirror`), і підхоплює стан системи навіть після перезапуску.

## Вимоги (Залежності)
Для роботи програми потрібні наступні пакети:
* `python3`
* `python3-gi` (PyGObject для роботи з інтерфейсом GTK3)
* `wlr-randr` (для нативного дублювання)
* `wl-mirror` (для дублювання у вікні)

> [!WARNING]
> **Сумісність:** Нативне дублювання працює тільки на справжніх `wlroots`-композиторах (наприклад, Sway, Wayfire, River, Labwc, MangoWM). Воно **НЕ працюватиме** на сучасних версіях Hyprland (оскільки розробник видалив підтримку протоколу `wlr-randr`), а також на GNOME чи KDE Plasma. Однак функція "Дублювання у вікні" (`wl-mirror`) працюватиме на Hyprland без проблем.

## Запуск
Просто запустіть скрипт:
```bash
python3 wayland-mirror.py
```

## Як це працює
В основі лежить парсинг виводу `wlr-randr`. При дублюванні програма динамічно зчитує координати екрана-джерела і фізично накладає на них цільовий монітор, застосовуючи апаратне масштабування. Крім того, перед цим вона зберігає його оригінальні координати в `~/.config/wayland-mirror.json`, щоб при скасуванні дублювання автоматично повернути екран на його законне місце.
Налаштування (обрана мова та збережені позиції екранів) зберігаються у `~/.config/wayland-mirror.json`.
